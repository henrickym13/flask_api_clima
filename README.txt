OBS: Trata-se de uma aplicação de um site simples feito em flask,
para consumir a api de clima da openwathermap.

OBS2: Antes de executar a aplicação é preciso fazer um cadastro no site:
https://openweathermap.org;
- Realizando o cadastro você obtem uma chave da api para uso;
- A chave leva algum tempo para ser ativada no servidores dele;
- Caso tente executar a aplicação antes da chave ser ativada, será exibido um erro.

Depois de feito os procedimento anteriores:
1 - execute no terminal o comando: env\Scripts\activate;
2 - depois python run.py;
3 - agora pode usar a aplicação.
